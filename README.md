## MAAZZBX

基于全新架构的正中靶心小助手。
由 [MaaFramework](https://github.com/MaaXYZ/MaaFramework) 强力驱动！


## 功能介绍

目前仅有自动战斗的功能，暂不支持自动领体力。

## 启动方式
点击MaaPiCli.exe， 按照提示选择客户端，选择正中靶心挂机task，最后选择 6. Run tasks即可开始挂机
